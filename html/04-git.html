Exercise: Installing Git

  • Go to https://git-scm.com/downloads
  • Install appropriate version

Testing your install

In a terminal window:

  • git help
  • cd [appropriate directory]
  • git clone git@gitlab.disney.com:python_class/beginning_python_jan2020.git

Problems? Are you on VPN?

Congratulations

You now have Git installed, and a copy of our repo downloaded.

Git in 60 seconds

  • Git is local, Gitlab is the server
  • Your repo is the whole repo
  • Local changes must be affirmatively added
  • Changes updated in repo with commit
  • Always include a descriptive message
  • Local commits only uploaded when pushed
  • Remote changes only downloaded when pulled or fetched

Daily-Use Commands

  • git status
  • git pull
  • git push
  • git add <filename>
  • git commit [-a] -m "<message>"
